Czym jest HTML i CSS?

HTML (HyperText Markup Language) jest podstawowym językiem służącym do tworzenia struktury stron internetowych. Dzięki niemu możemy definiować nagłówki, akapity, obrazy i linki.

CSS (Cascading Style Sheets) jest językiem odpowiedzialnym za wygląd strony. Pozwala na stylizację tekstu, kolorów, rozmiarów oraz układu elementów na stronie internetowej.
